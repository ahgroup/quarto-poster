# Products

The folders inside this folder contain final or near-final project products.

For an academic project, this often includes a manuscript. You may also
make presentations, posters, reports, websites, apps, or other deliverables.

The example folders are:

- `manuscript/`: an academic manuscript template.
- `presentation/`: example Quarto slides.
- `poster/`: a placeholder for poster materials.
- `report/`: an example HTML report.

Often you need a BibTeX reference file and a CSL style file for formatting
references. Files used across multiple products should usually go in `assets/`.
Files used by only one product can go in that product folder, often in a
`media/` subfolder.

You can add further folders. For example, if you have multiple presentations,
you might create a separate folder for each. If you write a blog post, you could
add a `blog-post/` folder.

AI tools can help draft and edit products, but all scientific claims,
interpretations, and citations need human review. Products should pull figures,
tables, and computed quantities from `results/` or documented product-specific
media folders rather than copying numbers by hand.
