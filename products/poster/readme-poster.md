# Poster

This folder can contain a poster created with Quarto or another tool.

There is currently no single user-friendly and flexible way to create
good-looking posters in Quarto. It is possible, but it may require some
adjustment.

If you want to try a Quarto/Typst poster, this template may be a useful starting
point:

https://github.com/quarto-ext/typst-templates/tree/main/poster
