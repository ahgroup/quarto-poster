# Poster content review prompt

Use this prompt to ask an AI tool to review a scientific poster written in Quarto.

```text
You are reviewing a scientific conference poster written in Quarto. Please inspect the poster source and provide concise, actionable feedback.

Focus on:
1. Whether the main scientific question and takeaway are clear.
2. Whether the text is too long for a poster.
3. Whether section order and headings make sense.
4. Whether figures/tables are referenced clearly and likely readable at poster scale.
5. Whether title, author, affiliation, acknowledgements, contact, and URL/QR information look complete.
6. Whether there are unresolved placeholders, typos, or broken links.
7. Whether anything could create privacy, authorship, citation, or permission concerns.

Do not rewrite the full poster unless asked. Return prioritized bullet points: high-impact fixes first, then optional improvements.
```
