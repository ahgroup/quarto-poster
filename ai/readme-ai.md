# AI assistance

This folder contains guidance for AI-supported work on the Quarto poster template.

- `ai-use-policy.md`: expectations for responsible AI use.
- `ai-use-log.md`: concise record of meaningful AI-assisted changes.
- `project-summary.yml`: AI-readable orientation summary. Human-facing docs are authoritative.
- `prompts/`: reusable prompt templates for reviewing poster content or maintaining the template.

Prompt files are plain Markdown templates. They are not scripts and do not run automatically. A human or AI tool can copy them into an AI chat, or tell an AI tool with repository access to read and apply a specific prompt.
